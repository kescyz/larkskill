# Lark Messenger

Send messages, interactive cards, and files as a bot. Manage group chats. All messages sent as bot identity, not user.

## Prerequisites

- LarkSkill MCP configured — run `npx larkskill install` or see https://larkskill-portal.pages.dev/setup
